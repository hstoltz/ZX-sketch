# ZX Sketch

ZX Sketch is a web-based graphical editor for the ZX Calculus.

Check out my application to the Unitary Fund:

- [Link to video](https://www.youtube.com/watch?v=q4Dd6d351VE)

Please feel free to contribute to the project or contact me at: Harry.Stoltz [at] nyu.edu

ZX Sketch uses [PyZX](https://github.com/zxcalc/pyzx?tab=readme-ov-file) for rewrites, and uses some minimal code from [ZXlive](https://github.com/zxcalc/zxlive).
